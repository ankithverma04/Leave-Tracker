About the Project

The Smart Attendance Leave Tracker is a comprehensive system for managing student leave applications and monitoring attendance. Designed for efficiency, it streamlines administrative tasks for educators and simplifies the leave application process for students.

  • Admin can approve or reject leave applications with comments.
  • Attendance file uploads with fine calculations for low attendance.
  • Students can view their leave history and application status.
  • Fine reductions for approved leave applications.
